# Screenshot Gallery

Every screenshot below is generated by CI from configs defined in
[`scripts/screenshots/configs.lua`](scripts/screenshots/configs.lua). Each
group shares one fixture file showing **all** color strings for that group.
Each screenshot enables only **one** config option, so you can see exactly
which strings it highlights (and which it doesn't).

If something looks off, click the indexed `[N]` link next to any screenshot
to open a pre-filled issue, or
[open an issue](https://github.com/catgoose/nvim-colorizer.lua/issues)
and reference the config key.
